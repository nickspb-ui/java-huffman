Требуется реализовать программным способом алгоритм кодирования-декодирования по методу Хаффмана. <br />
Схема кодирования информации в файле: 
- объект, представляющий дерево, построенное в ходе алгоритма и необходимое для получения кодовых последовательностей для каждого символа;
- длина закодированного сообщения в битах;
- закодированное сообщение.
